# PowerBi_dashboard_project

The dashboard helps HR teams understand:
- Employee Attrition
- Department-wise Performance
- Gender Distribution
- Employee Satisfaction
- Salary Analysis
- Attendance & Workforce Insights

- # Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

# Dataset Information
The dataset contains HR employee records including:
- Employee ID
- Age
- Gender
- Department
- Job Role
- Salary
- Attrition
- Experience
- Performance Rating

# Dashboard Features
- Interactive HR Dashboard
- Attrition Analysis
- Department-wise Employee Count
- Gender Diversity Insights
- Salary Distribution Charts
- Employee Performance Analysis

# Key Insights
- Highest attrition observed in Sales department
- Majority employees belong to age group 25–35
- Research & Development department has highest workforce
- Male employees are higher compared to female employees
